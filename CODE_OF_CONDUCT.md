# Code of Conduct

## Our Standards

Participants are expected to:

- Communicate respectfully and directly.
- Focus technical disagreement on facts, behavior, and tradeoffs.
- Avoid harassment, threats, discrimination, and personal attacks.
- Respect privacy and confidentiality, especially around credentials, logs, prompts, and source code.

## Scope

This code of conduct applies to project discussions, issues, reviews, documentation, and support channels where this project is maintained.

## Enforcement

Project maintainers may remove comments, decline contributions, or restrict participation when behavior undermines a safe and productive engineering environment.

For private deployments, the system owner or repository maintainer is responsible for enforcement.
